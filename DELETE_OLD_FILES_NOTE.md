# Catatan Clean Slate

```txt
index.html
assistant.html
assistant/
api/odoo.js
scripts/check-html.js
apps/web
apps/engine
render.yaml
wrangler.toml
```

Untuk repo baru yang benar-benar kosong, catatan ini bisa diabaikan.
